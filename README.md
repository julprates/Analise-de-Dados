# Compilador Automático de Dados em Excel - VBA

Projeto desenvolvido em VBA para automatizar a compilação das escalas de colaboradores de diferentes lojas. Cada loja possui arquivo próprio com as escalas de seus respectivos funcionários e para analisar indicadores era necessário manualmente compilar todas as escalas em um único arquivo. Com a automação, reduzi o tempo desse processo de consolidação de dados.

## Funcionalidades
- Importação automática de arquivos
- Consolidação de dados
- Atualização dinâmica
- Automação de processos manuais

## Tecnologias utilizadas

- Excel VBA (macros)
*É possível transformar esse código em Python, porém a tecnologia escolhida visou facilitar a utilização de colaboradores no setor que não possuem conhecimento em python. 

## Como usar
1. Abra o arquivo `Compilado.xlsm`
2. Habilite as macros
3. Clique no botão de Rodar Compilado
4. Selecione a pasta que contém os arquivos com as escalas
5. Acompanhe na barra abaixo o status do processo
6. Observe no Painel se todas as lojas foram compiladas
7. Salve o arquivo pelo botão do Painel para garantir padronização do nome e data do compilado

## Informações Importantes
O código foi desenvolvido para processar arquivos que seguem um padrão específico de estrutura e organização dos dados. Arquivos com layouts diferentes podem não ser reconhecidos corretamente pela automação. No entanto, o projeto pode ser facilmente adaptado para trabalhar com diferentes planilhas, colunas, linhas ou estruturas de dados através da edição do código VBA.
